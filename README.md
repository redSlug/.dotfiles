# .dotfiles

## Setup
- export repo_url=https://github.com/redSlug/.dotfiles.git
- `git clone $repo_url && cd .dotfiles && ./install`

## Add new dotfiles using dotbot
- commit file to repo, `install.conf.yaml`, and `./install` to create symlinks
